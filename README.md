# AI Sentiment Analyzer

An AI-powered web application that analyzes text sentiment using Natural Language Processing (NLP).

## Features
- Detects Positive and Negative sentiment
- User-friendly web interface
- Real-time sentiment prediction

## Technologies Used
- Python
- Flask
- Hugging Face Transformers
- HTML
- CSS

## Project Outcome
This project analyzes user-entered text and predicts whether the sentiment is positive or negative using a pre-trained NLP model.
